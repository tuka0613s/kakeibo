# 爆速家計簿

「テンキーで金額入力 → カテゴリタップで即登録」を核とした、入力速度最優先の家計簿 PWA。

**→ [今すぐ使う](https://tuka0613s.github.io/kakeibo/)**

---

## 特徴

- **1タップ登録** — 金額を打ってカテゴリをタップするだけ。登録まで2秒以内
- **完全オフライン対応** — Service Worker によりネットなしで動作
- **iPhoneホーム画面に追加** — App Store 不要。Safari から PWA インストール
- **データはすべて端末内** — localStorage に保存。外部サーバー通信なし

---

## 画面構成

| 画面 | 内容 |
|---|---|
| 入力 | テンキー・カテゴリグリッド・直近履歴 |
| カレンダー | 月カレンダー・月次サマリー・取引一覧 |
| 統計 | 直近6ヶ月棒グラフ・カテゴリ別支出・貯蓄率 |
| 設定 | カテゴリ管理・予算・初期残高・CSVエクスポート |

---

## インストール方法（iPhone）

1. Safari で **https://tuka0613s.github.io/kakeibo/** を開く
2. 下部の共有ボタン（□↑）をタップ
3. **「ホーム画面に追加」** を選択
4. アイコンからアプリとして起動

---

## 主な機能

### 入力画面
- テンキー常時表示（0–9・⌫・C の12キー）
- 支出 / 収入の切り替え
- カテゴリタップで即登録
- レイアウト A/B 切り替え（テンキーの位置を上下で変更可）
- 月次予算の進捗バー表示
- 直近6件を1行サマリーで表示・タップで編集

### カレンダー画面
- 月カレンダー（日別支出額・収入有無を表示）
- 月次サマリー（収入・支出・繰越金・残高）
- 初期表示はその月の全取引一覧
- 日付タップで絞り込み、再タップで月全体に戻る

### 統計画面
- 当月の収入・支出・貯蓄サマリー
- 直近6ヶ月の収入 vs 支出 棒グラフ
- カテゴリ別支出 横バーグラフ（上位6件）
- 月次貯蓄率バー

### 設定画面
- カテゴリ管理（追加・編集・削除・並び替え）
- 月次予算・アラート閾値（%）設定
- 初期残高設定（繰越金計算の起点）
- CSV エクスポート（今月 / 全件、Excel 対応）

## 技術スタック

| レイヤ | 技術 |
|---|---|
| フロントエンド | HTML / CSS / Vanilla JS（ビルドツールなし） |
| オフライン | Service Worker（Cache First） |
| データ永続化 | localStorage |
| 配布 | PWA + GitHub Pages |

ファイル構成：

```
kakeibo/
├── index.html      # アプリ本体（CSS・JS すべて内包）
├── manifest.json   # PWA マニフェスト
├── sw.js           # Service Worker
├── SPEC.md         # 機能仕様書
├── DESIGN.md       # 技術設計書
└── PROGRESS.md     # 開発進捗
```

## ローカルで動かす

ビルド不要。任意の静的サーバーで起動するだけ。

```bash
# Python を使う場合
python -m http.server 8080

# Node.js を使う場合
npx serve .
```

ブラウザで `http://localhost:8080` を開く。
