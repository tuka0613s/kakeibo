# 爆速家計簿

「テンキーで金額入力 → カテゴリタップで即登録」を核とした、入力速度最優先の家計簿 PWA。

**→ [今すぐ使う](https://tuka0613s.github.io/kakeibo/)**

---

## インストール（iPhone）

1. Safari で https://tuka0613s.github.io/kakeibo/ を開く
2. 下部の共有ボタン（□↑）→ **「ホーム画面に追加」**
3. ホーム画面のアイコンからアプリとして起動

ネット接続なしでも動作します（Service Worker によるオフラインキャッシュ）。

---

## 主な機能

### 入力画面
- テンキー常時表示（iOSのデフォルトキーボードは出ない）
- カテゴリをタップするだけで即登録
- レイアウト A/B 切り替え（テンキーを上/下に配置）
- 0キー位置切り替え（右端 ↔ 9の下）。下モード時は `00`・`−`/`+`/`=` 演算キーが追加
- 月次予算の進捗バー（設定時のみ表示）
- 直近6件を1行サマリーで表示

### カレンダー画面
- 月カレンダー（日別支出額・収入有無を表示）
- 月次サマリー（収入・支出・繰越金・残高）
- 初期表示はその月の全取引一覧
- 日付タップで絞り込み、再タップで月全体に戻る

### 統計画面
- 月次 / 年次の切り替えとナビゲーション
- 選択期間の収入・支出・貯蓄サマリー・貯蓄率バー
- 収入 vs 支出 棒グラフ（月次: 直近6ヶ月 / 年次: 1〜12月）
- カテゴリ別支出 横バーグラフ（上位8件）
- カテゴリバーをタップで推移グラフ＋取引履歴の詳細表示

### 設定画面
- カテゴリ管理（追加・編集・削除・並び替え）
- 月次予算・アラート閾値（%）設定
- 初期残高設定（繰越金計算の起点）
- CSV エクスポート（今月 / 全件・Excel対応）・CSV インポート
- 全データ消去

## 技術構成

| レイヤ | 技術 |
|---|---|
| フロントエンド | HTML / CSS / Vanilla JS（ビルドツールなし） |
| オフライン | Service Worker（Cache First） |
| データ永続化 | localStorage |
| 配布 | PWA + GitHub Pages |

単一ファイル構成（`index.html` に CSS・JS すべて内包）。Mac 不要、Windows PC のみで開発。

---

## ローカルで動かす

```bash
# Python の場合
python -m http.server 8080

# Node.js の場合
npx serve .
```

`http://localhost:8080` をブラウザで開く。
