# 爆速家計簿 機能仕様書

## コンセプト
「テンキーで金額入力 → カテゴリタップで即登録」を核とする、入力速度最優先の家計簿 PWA。  
対象端末：iPhone 17（iOS Safari）

---

## 画面構成

### 1. 入力画面（メイン）

#### レイアウト
ヘッダー右上のボタンで 2 種類を切り替え可能。設定は localStorage に永続保存。

| | Layout A（テンキー上） | Layout B（テンキー下） |
|---|---|---|
| 順序 | 金額 → テンキー → カテゴリ → メモ → 履歴 | 金額 → メモ → 履歴 → カテゴリ → テンキー |
| 特徴 | 入力→タップの動線が短い | 画面全体を使って履歴が見やすい |

#### 機能
| 機能 | 仕様 |
|---|---|
| 金額表示 | 入力中は強調色（インディゴ）で表示。¥0 は薄色 |
| テンキー | 0–9、00、⌫（1桁削除）、C（全消去）、.（小数点）、→（カテゴリ促し） |
| 支出/収入切替 | セグメントコントロール。切替でカテゴリ一覧も切り替わる |
| カテゴリ選択 | タップした瞬間に登録完了。金額0の場合はエラートースト |
| メモ | 省略可。タップでプロンプト入力 |
| 直近履歴 | 最新6件を表示。タップで編集シート（金額修正 / 削除） |

#### 支出カテゴリ（デフォルト12件）
食費・外食・カフェ・交通費・日用品・医療・衣服・娯楽・通信費・光熱費・家賃・その他

#### 収入カテゴリ（デフォルト6件）
給与・副業・ボーナス・投資・臨時収入・その他

### 2. カレンダー画面

| 要素 | 仕様 |
|---|---|
| 月カレンダー | 7列グリッド。日付セルに当日支出（k/万単位）と収入有無（+マーク）を表示 |
| 月ナビゲーション | ‹ › ボタンで前後月に移動 |
| 月次サマリーカード | 収入・支出・合計（当月収支）・繰越金（前月残高）・残高（合算）を表示 |
| 日別詳細 | 日付タップで当日の取引リストを展開。タップで編集シートを開く |
| 色分け | 日曜＝赤、土曜＝インディゴ、今日＝インディゴ背景 |

---

### 3. 統計画面

| 要素 | 仕様 |
|---|---|
| 切替 | 月次 / 年次タブ |
| サマリー | 収入・支出・貯蓄 の3項目 |
| 月次推移グラフ | 直近6ヶ月の収入（緑）vs 支出（赤）の縦棒グラフ |
| カテゴリ別支出 | 横バーチャートで上位カテゴリを表示 |
| 貯蓄率 | プログレスバー + パーセント表示 |

---

### 4. 設定画面

| 項目 | 仕様 | 状態 |
|---|---|---|
| カテゴリ編集・並び替え | 一覧表示・長押しで並び替え | 未実装 |
| カテゴリ追加 | 名前 + 絵文字で新規作成 | 未実装 |

| 予算アラート閾値 | 80%（デフォルト）。変更可 | 未実装 |
| 固定費設定 | 毎月自動登録する取引の設定 | 未実装 |
| CSVエクスポート | 月単位 or 全件でダウンロード | 未実装 |
| iCloud バックアップ | 将来対応（PWA の制約上、現状は localStorage のみ） | 将来 |

## 非機能要件

| 項目 | 要件 |
|---|---|
| 対応環境 | iPhone 17 / iOS Safari（PWA ホーム画面追加） |
| オフライン | Service Worker でアセットをキャッシュし、ネットなしで動作 |
| データ保存 | localStorage（単一デバイス内永続化） |
| パフォーマンス | 入力→登録まで1タップ・体感遅延ゼロ |
| 開発環境 | Windows PC（Mac 不要）。Vite 等ビルドツールなし・バニラ HTML/JS |
