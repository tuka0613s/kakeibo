# 爆速家計簿 機能仕様書

## コンセプト
「テンキーで金額入力 → カテゴリタップで即登録」を核とする、入力速度最優先の家計簿 PWA。

- **対象端末：** iPhone（iOS Safari）
- **公開URL：** https://tuka0613s.github.io/kakeibo/

---

## 画面構成

### 1. 入力画面（メイン）

#### レイアウト切り替え
ヘッダー右上のボタンで 2 種類を切り替え可能。設定は localStorage に永続保存。

| | Layout A（デフォルト） | Layout B |
|---|---|---|
| 順序 | 金額 → テンキー → カテゴリ → メモ → 履歴 | 金額 → メモ → 履歴 → カテゴリ → テンキー |
| 特徴 | 入力→タップの動線が短い | 履歴が見やすい |

ブラウザモード（ホーム画面未追加）の Layout B では、`hist-wrap` の `min-height` を解除してテンキーを常時表示する。

#### 各要素の仕様

| 要素 | 仕様 | 状態 |
|---|---|---|
| 金額表示 | 入力中はインディゴで強調。¥0 は薄色 | ✅ |
| テンキー | 0–9・⌫（1桁削除）・C（全消去）の12キー | ✅ |
| 予算進捗バー | 予算設定時のみ表示。緑→橙→赤で超過を視覚化 | ✅ |
| 支出/収入切替 | セグメントコントロール。カテゴリ一覧も連動 | ✅ |
| カテゴリ選択 | タップで即登録。金額0の場合はエラートースト | ✅ |
| メモ | 省略可。タップでボトムシートが開き入力（Enterキーで確定） | ✅ |
| 直近履歴 | 最新6件を1行サマリーで表示。タップで編集画面へ | ✅ |

#### テンキーレイアウト
```
7  8  9  ⌫
4  5  6  C
1  2  3  0
```

#### デフォルトカテゴリ

**支出（12件）：** 食費・外食・カフェ・交通費・日用品・医療・衣服・娯楽・通信費・光熱費・家賃・その他

**収入（6件）：** 給与・副業・ボーナス・投資・臨時収入・その他

### 2. カレンダー画面

| 要素 | 仕様 | 状態 |
|---|---|---|
| 月カレンダー | 7列グリッド。日付セルに支出額（k/万単位）と収入有無（+）を表示 | ✅ |
| 月ナビゲーション | ‹ › ボタンで前後月に移動 | ✅ |
| 月次サマリーカード | 収入・支出・合計・繰越金・残高を txns から動的計算 | ✅ |
| 取引リスト（初期） | 表示時はその月の全取引を新しい順で一覧表示 | ✅ |
| 日別絞り込み | 日付タップで当日のみ表示。再タップで月全体に戻る | ✅ |
| 即時反映 | 編集・削除後にその場でリストを再描画 | ✅ |
| 色分け | 日曜＝赤・土曜＝インディゴ・今日＝インディゴ背景 | ✅ |

### 3. 統計画面

#### 期間切り替え
ヘッダーのセグメントで月次 / 年次を切り替え。切り替えに連動してナビゲーションと全グラフが更新される。

| モード | ナビゲーション | サマリー対象 | 棒グラフ |
|---|---|---|---|
| 月次 | ‹ 年月 › で前後月に移動 | 選択月 | 選択月を末尾とした直近6ヶ月 |
| 年次 | ‹ 年 › で前後年に移動 | 選択年の合計 | 選択年の1〜12月 |

#### 各要素の仕様

| 要素 | 仕様 | 状態 |
|---|---|---|
| サマリー | 収入・支出・貯蓄を選択期間で動的計算 | ✅ |
| 推移グラフ | 収入/支出の縦棒グラフ（月次: 6ヶ月 / 年次: 12ヶ月） | ✅ |
| カテゴリ別支出 | 選択期間の支出上位8件を横バーチャートで表示 | ✅ |
| 貯蓄率 | プログレスバー + パーセント（収入に対する貯蓄割合） | ✅ |
| カテゴリ詳細 | カテゴリバーをタップで詳細オーバーレイを表示 | ✅ |

#### カテゴリ詳細オーバーレイ

カテゴリバーをタップすると全画面オーバーレイで以下を表示する。

| 要素 | 仕様 |
|---|---|
| サマリー | 合計・月平均・件数（選択期間内） |
| 推移グラフ | 月次モードは直近6ヶ月、年次モードは選択年の12ヶ月 |
| 取引履歴 | 該当カテゴリの取引を新しい順で最大30件表示。タップで編集可 |

### 4. 取引編集画面

履歴またはカレンダーの取引をタップすると全画面で開く。

| 要素 | 仕様 | 状態 |
|---|---|---|
| 支出/収入切替 | セグメントコントロール | ✅ |
| 金額入力 | カスタムテンキー（メイン画面と同一・iOS キーボード不使用） | ✅ |
| カテゴリ | グリッド選択（支出/収入に連動） | ✅ |
| メモ | テキスト入力フィールド | ✅ |
| 日付 | date ピッカー | ✅ |
| 保存・削除 | ヘッダーボタンとフッターボタン両方から操作可 | ✅ |

### 5. 設定画面

| セクション | 項目 | 仕様 | 状態 |
|---|---|---|---|
| カテゴリ | カテゴリを編集・並び替え | 一覧表示・↑↓で並び替え | ✅ |
| カテゴリ | カテゴリを追加 | 絵文字＋名前で新規作成 | ✅ |
| 予算 | 月次予算 | 金額設定。入力画面に進捗バー表示 | ✅ |
| 予算 | 予算アラート閾値 | デフォルト80%。超過時にトースト | ✅ |
| 残高 | 初期残高 | 記録開始前の口座残高。繰越金計算の起点 | ✅ |
| 定期支出 | 固定費を設定 | 毎月自動登録する取引（未実装・グレーアウト） | ⬜ |
| データ | CSV エクスポート / インポート | エクスポート：今月 or 全件・BOM付き。インポート：CSV 追記・エラー行スキップ | ✅ |
| データ | デモデータを登録 | 2024〜2026年の約400件。既存データありの場合は確認シート経由 | ✅ |
| データ | 全データを消去 | 確認シートを経由して txns・nextId をリセット | ✅ |
| データ | iCloudバックアップ | 将来対応（未実装・グレーアウト） | 🔮 |
| アプリ情報 | アップデートを確認・強制更新 | キャッシュ削除 → SW解除 → 強制リロード | ✅ |

## 非機能要件

| 項目 | 要件 | 状態 |
|---|---|---|
| 対応環境 | iPhone / iOS Safari（PWA） | ✅ 実機確認済み |
| オフライン | Service Worker でアセットをキャッシュ | ✅ |
| データ保存 | localStorage（単一デバイス内永続化） | ✅ |
| 入力速度 | カテゴリタップで即登録・体感遅延ゼロ | ✅ |
| キーボード | カスタムテンキー使用・iOS デフォルトキーボードは出さない（メモのみ text input） | ✅ |
| レイアウト | モバイルで全画面表示・デスクトップでモックフレーム表示 | ✅ |
| Safe Area | Dynamic Island / ホームインジケーター対応 | ✅ |
| 開発環境 | Windows PC のみ（Mac 不要・ビルドツールなし） | ✅ |
