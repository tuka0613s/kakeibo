# 爆速家計簿 機能仕様書

## コンセプト
「テンキーで金額入力 → カテゴリタップで即登録」を核とする、入力速度最優先の家計簿 PWA。  
対象端末：iPhone 17（iOS Safari）

---

## 画面構成

### 1. 入力画面（メイン）

#### レイアウト
ヘッダー右上のボタンで 2 種類を切り替え可能。設定は localStorage に永続保存。

| | Layout A（テンキー上） | Layout B（テンキー下） |
|---|---|---|
| 順序 | 金額 → テンキー → カテゴリ → メモ → 履歴 | 金額 → メモ → 履歴 → カテゴリ → テンキー |
| 特徴 | 入力→タップの動線が短い | 画面全体を使って履歴が見やすい |

#### 機能

| 機能 | 仕様 |
|---|---|
| 金額表示 | 入力中は強調色（インディゴ）で表示。¥0 は薄色 |
| テンキー | 0–9・⌫（1桁削除）・C（全消去）の12キー。`prompt()` 不使用 |
| 予算進捗バー | 月次予算設定時のみ表示。当月支出÷予算を色付きバーで表示（緑→橙→赤） |
| 支出/収入切替 | セグメントコントロール。切替でカテゴリ一覧も切り替わる |
| カテゴリ選択 | タップした瞬間に登録完了。金額0の場合はエラートースト |
| メモ | 省略可。タップでプロンプト入力（※要カスタムシート化） |
| 直近履歴 | 最新6件を1行サマリー（カテゴリ・メモ・日付・金額）で表示。タップで編集画面へ |

#### テンキーレイアウト

```
7  8  9  ⌫
4  5  6  C
1  2  3  0
```

#### 支出カテゴリ（デフォルト12件）
食費・外食・カフェ・交通費・日用品・医療・衣服・娯楽・通信費・光熱費・家賃・その他

#### 収入カテゴリ（デフォルト6件）
給与・副業・ボーナス・投資・臨時収入・その他

### 2. カレンダー画面

| 要素 | 仕様 |
|---|---|
| 月カレンダー | 7列グリッド。日付セルに当日支出（k/万単位）と収入有無（+マーク）を表示 |
| 月ナビゲーション | ‹ › ボタンで前後月に移動 |
| 月次サマリーカード | 収入・支出・合計（当月収支）・繰越金・残高を txns から動的計算 |
| 取引リスト（デフォルト） | 画面表示時はその月の全取引を新しい順で表示 |
| 日別絞り込み | 日付タップで当日の取引のみ表示。同じ日を再タップで月全体に戻る |
| 編集・削除の即時反映 | 編集・削除後、その場でリストを再描画（画面遷移不要） |
| 色分け | 日曜＝赤、土曜＝インディゴ、今日＝インディゴ背景 |

---

### 3. 統計画面

| 要素 | 仕様 |
|---|---|
| 切替 | 月次 / 年次タブ（年次は未実装） |
| サマリー | 当月の収入・支出・貯蓄を txns から動的計算 |
| 月次推移グラフ | 直近6ヶ月の収入（緑）vs 支出（赤）の縦棒グラフ。txns から動的生成 |
| カテゴリ別支出 | 当月の支出を横バーチャートで上位6件表示。txns から動的生成 |
| 貯蓄率 | プログレスバー + パーセント表示（収入に対する貯蓄の割合） |

---

### 4. 取引編集画面

| 要素 | 仕様 |
|---|---|
| 開き方 | 入力画面の履歴またはカレンダーの取引をタップ |
| 支出/収入切替 | セグメントコントロール |
| 金額入力 | カスタムテンキー（メイン画面と同一レイアウト）。iOS デフォルトキーボード不使用 |
| カテゴリ | グリッド選択（支出/収入に連動） |
| メモ | テキスト入力 |
| 日付 | date ピッカー |
| 保存・削除 | ヘッダーの「保存」「削除」ボタン、またはフッターの「保存する」ボタン |

---

### 5. 設定画面

| セクション | 項目 | 仕様 | 状態 |
|---|---|---|---|
| カテゴリ | カテゴリを編集・並び替え | 一覧表示・↑↓ボタンで並び替え | ✅ |
| カテゴリ | カテゴリを追加 | 絵文字＋名前で新規作成 | ✅ |
| 予算 | 月次予算 | 金額設定。入力画面に進捗バー表示 | ✅ |
| 予算 | 予算アラート閾値 | デフォルト80%。超過時にトースト | ✅ |
| 残高 | 初期残高 | 記録開始前の口座残高。繰越金計算の起点 | ✅ |
| 定期支出 | 固定費を設定 | 毎月自動登録する取引の設定 | ⬜ |
| データ | CSVエクスポート | 今月 or 全件。BOM付きCSV | ✅ |
| データ | iCloudバックアップ | 将来対応 | 🔮 |

## 非機能要件

| 項目 | 要件 |
|---|---|
| 対応環境 | iPhone 17 / iOS Safari（PWA ホーム画面追加） |
| オフライン | Service Worker でアセットをキャッシュし、ネットなしで動作 |
| データ保存 | localStorage（単一デバイス内永続化） |
| パフォーマンス | 入力→登録まで1タップ・体感遅延ゼロ |
| キーボード | カスタムテンキーを使用。iOS デフォルトキーボードは出さない |
| 開発環境 | Windows PC（Mac 不要）。Vite 等ビルドツールなし・バニラ HTML/JS |
